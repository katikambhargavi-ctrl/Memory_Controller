# 8×8 RAM Design

## Overview

This project implements an **8×8 Random Access Memory (RAM)** using Verilog HDL. The RAM consists of **8 memory locations**, each capable of storing **8 bits** of data. It supports synchronous read and write operations controlled by a clock signal.

## Features

* 8 memory locations with 8-bit data width.
* 3-bit address input for selecting memory locations.
* Synchronous write operation using a write enable (`we`) signal.
* Synchronous read operation on the rising edge of the clock.
* Memory is initialized to zero at the start of simulation.

## Files

* `ram8x8.v` – Verilog source code for the 8×8 RAM.
* `README.md` – Project documentation.
